# Battle-of-the-Boxes---COP-3003

  A roguelike dungeoncrawler (innovative, i know...) where the user plays as a box trying to escape an Amazon warehouse. During their escape, they have to battle zombie-like workers and finally defeat Jeff Bezos to win.
